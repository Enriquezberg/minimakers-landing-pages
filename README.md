# MiniMakers GT - Landing Pages

Landing pages profesionales para productos de MiniMakers GT.

## 🔗 Sitio web
https://minimakersgt.com

## 🌐 GitHub Pages
https://enriquezberg.github.io/minimakers-landing-pages/

## 📁 Estructura del proyecto
minimakers-landing-pages/
├── README.md
├── CLAUDE_CONTEXT.md
├── mini-panqueques-25/
│   └── index.html
└── [otros-productos]/
└── index.html
## 🛠️ Stack tecnológico

- HTML5 + CSS3 + JavaScript Vanilla
- Alpine.js (calculadoras)
- Swiper.js (galerías)
- GLightbox (videos)
- AOS (animaciones)
- Font Awesome (iconos)

## 🚀 Crear nueva landing
```bash
cp -r mini-panqueques-25 nombre-producto
code nombre-producto/index.html
git add .
git commit -m "Nueva landing: nombre-producto"
git push
```
